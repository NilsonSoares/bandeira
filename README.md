# BANDEIRA

Projeto desenvolvido durante o curso de Flutter pela Raro Academy, com o objetivo de reproduzir uma bandeira utilizando os *widgets* básicos do Flutter.

## Tecnologias Utilizadas

* Flutter
* Dart

## Conceitos Aplicados

* StatelessWidget
* MaterialApp
* Container
* Stack
* Positioned
